## 4)
Μελετήστε το παρακάτω πρόγραμμα και απαντήστε τις ερωτήσεις που ακολουθούν,

Υποθέστε ότι το μόνο σήμα που μπορεί να σταλεί στη διεργασία που εκτελεί το πρόγραμμα είναι το SIGALRM.

```c
static void handler (int sig) {

write(1, "Delta\n", 5);

}

#include <stdio.h>
#include <unistd.h>
#include <signal.h>
#include <string.h>
#include <errno.h>
#include <sys/time.h>

void handler(int signum) {
    printf("Alarm triggered!\n");
}

int main(int argc, char *argv[]) {
    int n;
    char buff[10] = {'\0'};

    struct sigaction action = {{0}};
    action.sa_handler = handler;
    sigaction(SIGALRM, &action, NULL);

    struct itimerval t = {{0}};
    t.it_value.tv_sec = 5;
    t.it_value.tv_usec = 0;      // Initialize microseconds to zero
    t.it_interval.tv_sec = 5;
    t.it_interval.tv_usec = 0;   // Initialize microseconds to zero

    setitimer(ITIMER_REAL, &t, NULL);

    while (1) {
        n = read(STDIN_FILENO, buff, 9);
        if (n > 0) {
            buff[n] = '\0';  // Null-terminate the string
            printf("%s\n", buff);
        } else if (n == 0) {
            printf("Alpha\n");
        } else if (errno == EINTR) {
            printf("Beta\n");
        } else {
            printf("Gamma\n");
        }
    }

    return 0;
}
```

#### - Πότε εκτυπώνεται το μήνυμα “Gamma\n”

#### - Πότε εκτυπώνεται το μήνυμα “Delta \n”

#### - Πότε εκτυπώνεται το μήνυμα “Alpha \n”

#### - Πότε εκτυπώνεται το μήνυμα “Beta \n”
#
## ANS:

Το πρόγραμμα:

- Ορίζει έναν χειριστή σήματος handler() για το σήμα SIGALRM.

- Ρυθμίζει έναν επαναλαμβανόμενο χρονοδιακόπτη (setitimer) να στέλνει SIGALRM κάθε 5 δευτερόλεπτα.

- Ο χειριστής handler τυπώνει "Delta\n" με write().

- Στη συνέχεια, σε έναν ατέρμονο βρόχο, καλεί read() για να διαβάσει από το standard input (STDIN_FILENO).

- Ανάλογα με την τιμή επιστροφής από read(), εμφανίζει διαφορετικά μηνύματα.

) Πότε εκτυπώνεται το μήνυμα Gamma\n

Gamma\n εκτυπώνεται όταν η read() αποτυγχάνει με κάποιο σφάλμα εκτός του EINTR.

- Το read() επιστρέφει -1 και η errno είναι διαφορετική από EINTR.

- Δηλαδή, σφάλματα όπως EBADF, EIO, EFAULT, κτλ.

- Αυτό θα συμβεί αν υπάρξει πρόβλημα με το αρχείο εισόδου (π.χ., stdin έχει κλείσει ή είναι κατεστραμμένο).

✅ Άρα: Όταν η read() επιστρέψει -1 και errno != EINTR.

2) Πότε εκτυπώνεται το μήνυμα Delta\n

Αυτό είναι το μήνυμα που τυπώνεται μέσα στον handler του σήματος SIGALRM.

- Ο χειριστής του σήματος SIGALRM τυπώνει "Delta\n" κάθε φορά που αυτό το σήμα λαμβάνεται.

- Λόγω του setitimer(), το σήμα SIGALRM αποστέλλεται κάθε 5 δευτερόλεπτα.

✅ Άρα: Κάθε 5 δευτερόλεπτα, όταν το σήμα SIGALRM φτάσει και εκτελεστεί ο handler.

3) Πότε εκτυπώνεται το μήνυμα Alpha\n

Η read() επιστρέφει 0 όταν ανιχνεύσει EOF (τέλος αρχείου).

- Αυτό μπορεί να συμβεί αν ο χρήστης κάνει EOF στο stdin, π.χ. με Ctrl+D (σε Unix/Linux).

- Το EOF δεν είναι σφάλμα. Απλώς δείχνει ότι δεν υπάρχει άλλο input.

✅ Άρα: Όταν ο χρήστης πατήσει Ctrl+D (ή γενικά stdin επιστρέψει EOF).

4) Πότε εκτυπώνεται το μήνυμα Beta\n

Η read() μπορεί να διακοπεί από ένα σήμα ενώ περιμένει για είσοδο.

- Αν κατά τη διάρκεια της read() έρθει ένα σήμα (SIGALRM), τότε η read() επιστρέφει -1 και η errno γίνεται EINTR.

- Σε αυτή την περίπτωση, το πρόγραμμα εκτυπώνει "Beta\n".

✅ Άρα: Όταν η read() διακοπεί από το σήμα SIGALRM, δηλαδή περίπου κάθε 5 δευτερόλεπτα, εφόσον δεν υπάρχει ενεργή είσοδος.

📌 Συνοπτικά:

Gamma\n	Όταν η read() αποτυγχάνει με errno ≠ EINTR

Delta\n	Κάθε 5 δευτερόλεπτα από τον handler του SIGALRM

Alpha\n	Όταν ο χρήστης πατήσει Ctrl+D (EOF στο stdin)

Beta\n	Όταν η read() διακοπεί από SIGALRM και errno == EINTR
